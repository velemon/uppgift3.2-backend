Om denna webbplats

Denna webbplats är skapad som en fristående frontend-applikation som kommunicerar med ett REST-baserat API med hjälp av Fetch API i JavaScript. Webbplatsens syfte är att hantera arbetserfarenheter genom att visa, lägga till och radera poster från en databas. Frontend-delen är byggd med HTML, CSS och JavaScript medan backend-delen är utvecklad med Node.js och Express. Datan lagras i en MongoDB-databas och hanteras med hjälp av Mongoose. MongoDB är en dokumentbaserad NoSQL-databas där information lagras som dokument istället för i traditionella tabeller. Det gör databasen flexibel och väl lämpad för webbapplikationer som arbetar med JSON-data.

Webbplatsen är uppdelad i tre undersidor:

  • Start (index.html) – visar alla lagrade arbetserfarenheter från databasen
  • Lägg till (add.html) – möjlighet att lägga till nya arbetserfarenheter
  • Om sidan (about.html) – information om webbplatsen och dess syfte

Webbplatsen använder Fetch API för att kommunicera med webbtjänsten och utföra olika CRUD-operationer:

  • GET – hämtar lagrade arbetserfarenheter
  • POST – lägger till nya arbetserfarenheter
  • DELETE – raderar befintliga poster

Syfte

Syftet med uppgiften är att:

  • Visa dina kunskaper inom NoSQL-databaser.
  • Kunna skapa och administrera MongoDB-databaser.
  • Kunna skapa en REST-baserad webbtjänst med NodeJs, Express och MongoDb.
